# 📕 Catálogo de Publicações do COBIT e Suas Aplicações

---

## 1. Introdução

O COBIT não é apenas um framework único, mas um **ecossistema de publicações** que apoiam diferentes áreas da governança de TI.

📌 Cada documento tem uma função específica:
- implementação  
- auditoria  
- controle  
- segurança  
- estratégia  

---

## 2. Classificação das Publicações COBIT

As publicações podem ser organizadas em categorias:

| Categoria | Objetivo |
|----------|--------|
| Estrutura principal | Definir o framework |
| Implementação | Aplicar governança |
| Auditoria | Avaliar processos |
| Controle | Detalhar práticas |
| Segurança | Proteger informação |
| Estratégia | Apoiar decisão |
